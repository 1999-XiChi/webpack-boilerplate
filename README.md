## About Yeoman

THE WEB'S SCAFFOLDING TOOL FOR MODERN WEBAPPS. 脚手架工具

1. 全局安装yeoman脚手架工具： `npm install -g yo`
2. 安装你需要的脚手架依赖包，如 `npm install generator-imooc-gulp`
3. 脚手架安装ing： `yo imooc-gulp`
